## React Weather Application

Simple React weather application that utilizes the OpenWeatherMap API.

### `React Framework`

JavaScript library for building interfaces.<br />
Download stable release 16.12.0 at [https://reactjs.org/](https://reactjs.org/).

### `OpenWeatherMap API`

Current and forecast weather data collection.<br />
Sign up for OpenWeatherMap API at [https://openweathermap.org/](https://openweathermap.org/).
